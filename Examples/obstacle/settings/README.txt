Salome_obstacle.hdf is the file of the salome project to create mesh


create_obstacle_mesh.py is the python script to run process building mesh of obstacle  

salome -t python create_obstacle_mesh.py is the comand to run chosen script in interative mode

exportMesh.py is auxilary statements to convert salome mesh format to openfoam or elmer one. 

auto_mesh_salome is the folder consisting of templates of parametrized mesh.
